# Roadmap

This document outlines the future development path for PacketFlow.

## Q3 2026
- **FlashList Migration**: Upgrade FlatLists to `@shopify/flash-list` for improved memory recycling in the device library and notifications.
- **Enhanced Topology Caching**: Implement advanced local SQL caching for massive enterprise topologies to allow instant offline loading.

## Q4 2026
- **Real-Time Collaboration**: Enable multi-user simultaneous editing on the same canvas using Supabase Realtime subscriptions.
- **Advanced Packet Simulation**: Integrate physical layer latency simulation (BGP routing visualizations).

## 2027
- **Tablet Optimization**: Dedicated iPad/Android tablet layouts.
- **Web Export**: Static HTML/Canvas export for sharing topologies via links.
